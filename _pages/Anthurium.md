# 火鶴(花燭) *Anthurium*

# 印尼的花燭都經歷過大亂交時代，數據很多都已不可考
# 就連南美的也一堆雜交
# 建議一律以*Anthurium* unknown 或是 *Anthurium* hybird 稱呼印尼來的未知花燭
# 不是sp. 不是sp. 不是sp. 很重要所以講三次

我個人沒有玩花燭類，但蠻多人私訊希望有花燭類列表所以就做了

名單由徐煒倫提供，並由本人整理再加上幾年有看過的進口種得出
